# OxPulse Chat — Marketing Strategy

## Как это работает

Продукт продаёт себя сам. Каждый звонок — это маркетинг.

```
Ты скидываешь ссылку oxpulse.chat/BEAR-7042 племяннице
  → Она открывает, вы говорите
  → Она спрашивает: "а я могу так же позвонить подружке в Таиланд?"
  → Она скидывает ссылку подружке
  → Подружка скидывает кому-то ещё
  → ...
```

Это единственная маркетинговая стратегия которая нужна на старте. Всё остальное — усиление этого эффекта.

## Кто наши пользователи

Люди, которым нужно позвонить за границу, а привычные способы не работают.

- Россия ↔ Европа (Telegram блокируют, Zoom лагает, VPN сложно)
- Россия ↔ США/Канада
- Россия ↔ Израиль, Грузия, Турция, Таиланд, Бали
- Любая страна с цензурой ↔ весь мир

**Это не "privacy-conscious developers". Это обычные люди** — мамы, бабушки, друзья, разбросанные по миру. Им не нужно знать что такое VLESS или E2EE. Им нужно нажать на ссылку и увидеть лицо.

## Почему это вирально

1. **Ссылка — это продукт.** Не нужно скачивать приложение, регистрироваться, ждать. Нажал — говоришь.
2. **Каждый звонок создаёт двух пользователей.** Тот кто скинул ссылку + тот кто её открыл.
3. **Второй пользователь сразу понимает ценность** — он только что поговорил. И хочет сделать то же самое с кем-то ещё.
4. **Работает там, где другие не работают.** Если у тебя заблокирован Telegram — ты расскажешь всем друзьям про то, что работает.

## Что нужно сделать чтобы это работало

### В продукте

- [ ] После звонка: кнопка "Создать свою комнату" (CTA для получателя)
- [ ] OG-превью при отправке ссылки в мессенджерах (красивая карточка)
- [ ] Лендинг понятен без слов: одна кнопка "Создать звонок", одно поле "Войти по коду"
- [ ] Работает на любом телефоне без установки (PWA)
- [ ] Максимально быстрый первый звонок (<5 секунд от клика до лица)

### Первые пользователи (сарафан)

- [ ] Скинуть 10 друзьям/знакомым кто звонит за границу
- [ ] Попросить каждого попробовать позвонить кому-то ещё
- [ ] Собрать фидбек: что непонятно, где застряли, что сломалось

### Усиление (когда сарафан уже работает)

- [ ] Короткое видео (30 сек): "Как позвонить за границу без VPN"
- [ ] Telegram-каналы диаспоры (русские в Германии, Израиле, Турции, Грузии)
- [ ] Yandex SEO: "видеозвонки без VPN", "позвонить за границу из России"
- [ ] Show HN (для GitHub stars и self-hosted аудитории)

### Не делаем (пока)

- Платную рекламу
- Партнёрства с организациями
- PR / пресс-релизы
- Сложные воронки
- Контент-маркетинг

## Метрика успеха

Одна: **сколько людей после первого звонка сами создали комнату и позвонили кому-то ещё.**

Если этот процент > 0 — продукт работает. Всё остальное — следствие.

## Риски

| Риск | Что делаем |
|------|-----------|
| РКН заблокирует VLESS | Fallback протоколы, смена домена маскировки |
| Никто не пересылает ссылку дальше | Значит UX плохой — чиним |
| Конкуренты скопируют | Open source + mesh + скорость |
